# User Stories

## Engineer

- As an engineer, I want a short ranked feed so I can quickly see what may affect my tools, skills, or career.
- As an engineer, I want each item to explain why engineers should care so I can decide whether to read the source.
- As an engineer, I want categories so I can focus on AI, developer tools, hardware, security, cloud, research, or career signals.

## Student

- As a student, I want suggested actions so I know what to learn or try next.
- As a student, I want non-hype language so I can separate useful signals from marketing.

## Technical Founder

- As a technical founder, I want startup and infrastructure signals so I can spot emerging tools and market shifts.

## Developer Working On The Project

- As a subagent, I want shared types so I can build my part without waiting for unfinished implementations.
- As an integration agent, I want ownership boundaries so I can merge branches with fewer conflicts.
